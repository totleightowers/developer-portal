# 

You can obtain AWS credentials to use the AWS CLI or other command line tools.

1. [Log in to the AWS Console](accessing-the-aws-console)

1. Select the account you wish to access programmatically

1. Choose the `Command line or programmatic access` options

1. Follow the instructions on screen to retrieve your credentials.
